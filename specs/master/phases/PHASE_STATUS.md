## 更新规则

- 进入 phase 时，将该 phase 状态改为 `in_progress`。
- 验收通过后，将状态改为 `completed`，并在 Completion evidence 中补充实际产物路径。
- 如果验收失败，将状态改为 `blocked`，并在当前 phase 的 checkpoint 或 gap report 中说明原因。
- 不允许在前置 phase 仍为 `planned`、`in_progress` 或 `blocked` 时启动依赖它的 phase。
